Fundamentação Teórica

Antes de escrevermos código, precisamos entender a natureza da ferramenta. SQL não é apenas sobre comandos; é sobre declaração e estrutura.

🗣️

Natureza Declarativa

SQL é uma linguagem declarativa. Diferente de Java ou C (procedurais), você diz ao banco o que quer ("Busque os clientes"), e o SGBD decide como buscar (otimização).

🛠️

SQL vs. MySQL

Não confunda a linguagem com a ferramenta.

SQL: O padrão ANSI/ISO (o idioma).
MySQL: O software SGBD que entende e executa o idioma.

Taxonomia dos Comandos

Entenda a diferença crucial entre definir a casa (Estrutura) e mobiliar a casa (Dados).

DDL - Data Definition Language

Gerencia a Estrutura (Metadados)

  • CREATE Cria objetos como tabelas, views e índices.
  • ALTER Modifica estruturas existentes (add colunas).
  • DROP Destrói estruturas inteiras (irreversível).

DML - Data Manipulation Language

Gerencia as Instâncias (Linhas)

  • INSERT Adiciona novos registros na tabela.
  • SELECT Recupera e filtra dados (o coração do SQL).
  • UPDATE Atualiza dados existentes.
  • DELETE Remove registros específicos.
